# JasunBag Astro Static Site

High-performance static website template for a B2B gift bag packaging manufacturer.

## Why Astro

- Static output, ideal for Cloudflare Pages
- Fast page loading for SEO
- Easy product category pages through data files
- Built-in sitemap support
- Good structure for Google indexing and AI answer engines

## Pages

- `/` - Home
- `/products/` - Product center
- `/products/[slug]/` - Dynamic product category pages
- `/customization/` - Custom logo and packaging services
- `/sustainability/` - Eco-friendly material positioning
- `/gallery/` - Product gallery
- `/blog/` - Buyer guides
- `/about/` - Company introduction
- `/contact/` - RFQ form

## SEO Files

- `public/robots.txt`
- `public/llms.txt`
- sitemap generated by `@astrojs/sitemap`
- schema added for Organization and Product pages

## Local Preview

Install dependencies first:

```bash
npm install
```

Start local development:

```bash
npm run dev
```

Build static site:

```bash
npm run build
```

## Cloudflare Pages

Build command:

```bash
npm run build
```

Build output directory:

```bash
dist
```

## Important

Before going live, update `astro.config.mjs` if the final domain is not `https://www.jasunbag.com`.
